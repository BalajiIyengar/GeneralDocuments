Tags can be added post committing also(at any point).Even after 6 months' time.

Tags: 
	Annotated
	Lightweight


	
		Annotated: 
			Contain the data of the committer,the data of commit et cetera
			$ git tag -a v1.4 -m 'my version 1.4'
			
			-a denotes annotation
			-m is the message

		Lightweight:
			Dont store any data


Pushing a tag to remote repo
 git push origin [tagname].

Pushing multiple tags at once
git push origin --tags


Tagging a previous commit from log
$ git log --pretty=oneline
$ git tag -a v1.2 -m 'version 1.2' 9fceb02



Setting up aliases for commands (type only the aliases instead of the command)

Some Examples:
$ git config --global alias.co checkout
$ git config --global alias.br branch
$ git config --global alias.ci commit
$ git config --global alias.st status
$ git config --global alias.unstage 'reset HEAD --'
$ git config --global alias.last 'log -1 HEAD'
